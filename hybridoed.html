hybridoed.forward module¶

hybridoed.forward.create_src_field(N, x, y, domain, omega)[source]¶

Create a source field with bilinear interpolation at the given (x, y) coordinates and compute its Fourier Series.

Parameters:

  • N (tuple of int) – The size of the source field array as (height, width).

  • x (float) – The x-coordinate of the source (can be a fractional value).

  • y (float) – The y-coordinate of the source (can be a fractional value).

  • domain (array-like) – The domain for the Fourier Series calculation.

  • omega (float) – The scaling factor for the Fourier Series.

Returns:

The computed Fourier Series of the source field.

Return type:

array

Notes

  • The source field uses bilinear interpolation to assign values at the four nearest grid points.

  • The Fourier Series is computed using the FourierSeries function (assumed to be defined elsewhere).

Example

>>> import jax.numpy as jnp
>>> N = (100, 100)
>>> x, y = 45.5, 76.3
>>> domain = jnp.linspace(0, 1, 100)
>>> omega = 2.0
>>> src = create_src_field(N, x, y, domain, omega)
>>> src.shape
(100,)
